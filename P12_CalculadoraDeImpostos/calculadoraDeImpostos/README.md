<h1 align="center">Calculadora de impostos</h1>
<h3 align="center">O projeto realiza um cálculo e apresenta o imposto a ser pago pelo usuário a partir da renda que ele insere.</h3>

- Dependências utilizadas nesse projeto: formik, yup, @emotion/react, @emotion/styled e @mui/material
- O valor do imposto apresentado não é real e nem exato, o objetivo principal desse projeto foi utilizar o Formik.
  
</p>
