<h1 align="center">Conversor de moedas</h1>
<h3 align="center">O projeto é um conversor de moedas onde o usuário escolhe o valor, a moeda que ele deseja converter, a moeda que ele deseja convertida e recebe o valor equivalente.</h3>

- Para buscar as moedas de diferentes países foi utilizada a api da [https://www.exchangerate-api.com/](https://www.exchangerate-api.com/)
- Para consumir a api foi utilizada o axios
  
</p>
